### Run Locally

To run **Jamms Hospital Web** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/rivalsolmons/JAMMS_HOSPITAL.git
```

Windows:

```bash
git clone https://github.com/rivalsolmons/JAMMS_HOSPITAL.git
```
